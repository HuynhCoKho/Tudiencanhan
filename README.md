# 📚 Từ Điển Cá Nhân / Personal Dictionary

## 🇻🇳 Giới thiệu

**Từ Điển Cá Nhân** là ứng dụng từ điển học tập cá nhân chạy trực tiếp trên trình duyệt web (**Chrome / Edge / Firefox...**) mà không cần cài đặt phần mềm.

Ứng dụng hỗ trợ tra cứu và quản lý từ vựng giữa:

- 🇻🇳 Tiếng Việt
- 🇬🇧 Tiếng Anh
- 🇨🇳 Tiếng Trung
- 🇯🇵 Tiếng Nhật
- 🇰🇷 Tiếng Hàn
- 🇷🇺 Tiếng Nga
- 🇹🇭 Tiếng Thái
- 🇫🇷 Tiếng Pháp
- 🇩🇪 Tiếng Đức
- 🇮🇹 Tiếng Ý
- 🇪🇸 Tiếng Tây Ban Nha
- 🇵🇹 Tiếng Bồ Đào Nha
- Và nhiều ngôn ngữ khác...

Không chỉ là một cuốn từ điển thông thường, đây là nơi bạn có thể xây dựng **"kho kiến thức ngôn ngữ của riêng mình"**.

### ✨ Điểm nổi bật

✅ Tự thêm / sửa / xóa mục từ  / nhập bộ từ điển từ nhiều nguồn khác nhau (*.BGL, LD2, JSON)

✅ Ghi chú nghĩa theo cách bản thân dễ hiểu nhất  

✅ Hỗ trợ thêm phiên âm IPA / Pinyin  

✅ Thêm hình ảnh minh họa cho các từ khó hình dung:

- 🌳 Cây cối
- 🍎 Trái cây
- 🍜 Món ăn
- 🪑 Vật dụng địa phương
- 🏠 Đồ dùng sinh hoạt
- Và bất kỳ nội dung nào bạn muốn ghi nhớ

✅ Chạy trực tiếp trên trình duyệt  

✅ Không cần cài đặt  

✅ Dữ liệu có thể tự quản lý  

✅ Phù hợp cho:

- Người học ngoại ngữ
- Giáo viên
- Học sinh – sinh viên
- Người tự học
- Người cần xây dựng bộ thuật ngữ chuyên ngành riêng

Nếu thấy hữu ích, mọi người có thể dùng để tự xây dựng một **từ điển học tập cá nhân** theo đúng nhu cầu của mình.

## 🇺🇸 Introduction

**Personal Dictionary** is a browser-based learning dictionary that runs directly on **Chrome / Edge / Firefox** without software installation.

The application supports vocabulary management between Vietnamese and multiple languages:

- English
- Chinese
- Japanese
- Korean
- Russian
- Thai
- French
- German
- Italian
- Spanish
- Portuguese
- And many others...

More than a traditional dictionary, it allows users to build their own **personal language knowledge system**.

### ✨ Key Features

✅ Add / edit / delete entries  

✅ Create custom notes using your own understanding  

✅ Add IPA / Pinyin pronunciation  

✅ Insert images for difficult concepts:

- Plants
- Fruits
- Food
- Local objects
- Daily-life items
- Any content you want to remember

✅ Runs directly in browser  

✅ No installation required  

✅ User-controlled data  

✅ Suitable for:

- Language learners
- Teachers
- Students
- Self-learners
- Professionals building terminology collections

If you find it useful, feel free to create your own personalized learning dictionary.
