# Tu dien ca nhan

Ban GitHub Pages tinh cua tu dien ca nhan.

Trang chay truc tiep tai:

https://huynhcokho.github.io/Tudiencanhan/

## Luu y du lieu

- Du lieu goc nam o `data/dictionary.json`.
- Tu moi, sua, xoa tren trang web duoc luu trong trinh duyet bang localStorage.
- Dung nut `Xuat JSON` de sao luu du lieu ca nhan.
- Dung nut `Nhap JSON` de phuc hoi hoac them bo tu dien moi.

Neu muon trang co san bo tu dien khi moi mo, thay file `data/dictionary.json` bang file JSON da xuat tu app local.
